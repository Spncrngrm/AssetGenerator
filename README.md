# AssetGenerator

AssetGenerator is a machine learning project focused on generating **64x64 top-down pixel art assets** in a **GBA-era monster-collector RPG style** for use in a 2D game environment.

The long-term goal of this project is to train a custom model capable of producing consistent overworld assets such as props, terrain elements, vegetation, structures, and other environment art that fit a unified visual style.

## Current Status

This project is currently in early development.

Initial work includes:
- setting up the Python environment
- preparing the project structure
- building and curating training datasets
- experimenting with small image-generation models
- developing a workflow for training, testing, and sampling generated assets

## Project Goals

- Generate original **64x64 top-down pixel art**
- Maintain a consistent visual style across generated assets
- Support use in a custom game project
- Build a reproducible training pipeline for future improvement and expansion
